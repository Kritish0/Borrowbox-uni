# BorrowBox Uni

BorrowBox Uni is a web-based student rental platform prototype designed to help university students rent, buy, and sell essential items such as stationery, furniture, room items, sports gear, and notes.

## Features
- Login screen
- Search functionality
- Rent, buy, and sell item flows
- Category browsing
- Cart system
- Listings section
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to explore a practical solution for students who need affordable access to temporary or essential items during university.
